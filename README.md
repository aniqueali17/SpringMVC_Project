<h1 align="center" id="title">Mini-SurveyMonkey</h1>

<p id="description">Mini-SurveyMonkey is a lightweight survey application designed for creating, distributing, and analyzing surveys. This project was developed as part of the SYSC 4806 course.</p>

<h2>Current State</h2>
<p>MileStone 1:</p>

-  Implemented basic user creation with automatically generated IDs.

-  Set up the User entity and repositories.
  
-  Built a Thymeleaf-based UI for user creation.

-  Completed deployment testing and successfully deployed the project to Azure.

<h2>Installation Steps:</h2>

<p>1. Clone the repository</p>

```
git clone https://github.com/aniqueali17/SpringMVC_Project.git
```

<p>2. Run the main file</p>

<p>3. Open the application in your browser - go to: http://localhost:8080/users</p>
